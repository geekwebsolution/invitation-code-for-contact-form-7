== Description ==

Invitation Code for Contact Form 7 helps you to find confirm subscribers and connect subscribers to your buisness.

Using this plugin, you can create an invitation code that you can use in contact form 7 to get some limited and genuine leads.

The invitation code for the contact form 7 plugin is very easy to use. Just install and activate the plugin and generate your combinations of codes. And share that code with your contacts. The form remains incomplete until any user can enter the given invitation code. Once users complete all the details, including the invitation code, they are all considered genuine leads.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Contact form 7-> Invitation Codes
